# W1D3 - Data Loading, Cleaning & Inspection

## Objective
Perform data loading, cleaning, and inspection on a real-world startup funding dataset using Pandas and NumPy.

## Dataset
Indian Startup Funding Dataset

Records:
- 3044 rows
- 10 columns

## Tasks Completed

- Loaded CSV dataset using Pandas
- Inspected dataset structure and information
- Identified missing values
- Cleaned column names
- Handled missing categorical values
- Handled missing investment values
- Converted investment amount into numeric format
- Checked and removed duplicate records
- Performed NumPy array operations
- Exported cleaned dataset

## Technologies Used

- Python
- Pandas
- NumPy

## Output

Generated file:

`cleaned_startup_funding.csv`
